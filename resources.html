Excel Keyboard Shortcuts

  • Ctrl + C: Copy
  • Ctrl + V: Paste
  • Ctrl + Z: Undo
  • Ctrl + Arrow Keys: Jump to data edges
  • Ctrl + Shift + L: Toggle filters
  • Alt + E, S, V: Paste Special
  • Ctrl + T: Create Table
  • F4: Repeat last action
  • Alt + F1: Insert chart
  • Ctrl + Shift + "+": Insert cell
